2.1.1 미리보는 인덱스 튜닝
데이터를 찾는 두가지 방법 홍길동 찾기

  1. 1학년 1반부터 6학년 맨 마지막 반까지 모두 돌기 (테이블 전체 스캔)

홍길동이 많다면 전자가 빠르고, 몇 안되면 후자가 빠르다

인덱스 튜닝의 두 가지 핵심요소 인덱스는 큰 테이블에 소량 데이터를 검색할 때 사용 온라인 트랜잭션 처리 시스템에서는 소량데이터를 주로 검색하므로 인덱스 튜닝 중요

인덱스 튜닝 2가지 방법

  1. 인덱스 스캔 과정에서 발생하는 비효율을 줄임 = 인덱스 스캔 효율화 튜닝
  2. 테이블 액세스 횟수 줄이기 = 랜덤 엑세스 최소화 튜닝
